## About the Project

This is an interactive **Streamlit** application designed to process heart rate data from CSV files, conduct a structured clinical assessment (anamnesis) with the user, and generate a cardiovascular risk stratification report based on decision logic. 

> ⚠️ **Disclaimer:** This software is for educational and demonstration purposes only. It does not replace professional medical advice, diagnosis, or treatment.

---

## Features

* **Data Processing:** Reads and cleans physiological heart rate data directly from uploaded `.csv` files.
* **Interactive Anamnesis:** A dynamic questionnaire built with Streamlit to gather user health habits and history.
* **Risk Stratification:** Evaluates the combined data to output a personalized cardiovascular risk assessment.
* **Clean UI:** Simple, modern, and user-friendly interface.

---

##  Technologies

* **Python** (Core logic)
* **Streamlit** (Web Interface)
* **Pandas** (Data Manipulation)
* **Matplotlib.pyplot** (Data graphical representation)
